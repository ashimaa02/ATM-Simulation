# ATM Simulation Project

## Description
This project is a Python-based ATM simulation system. It allows users to perform banking operations like checking balance, depositing money, withdrawing money, and viewing transaction history.

The project is built using a structured approach with separate modules for better readability and maintainability.

---

## Features
- Display Balance
- Deposit Money
- Withdraw Money
- Transaction History
- Input Validation

---

## Project Structure
- main.py → Menu-driven program
- services → Business logic
- models → Data handling
- utils → Helper functions

---

## How to Run
Run the following command:
python main.py
